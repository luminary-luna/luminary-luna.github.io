The Mirror's Malevolence

Tool(s): ChatGPT, Firefly

Project Description: “The Mirror’s Malevolence” is a mock up of a book cover, conceptualized using purely AI. All text used on the cover itself was generated through ChatGPT, only being given a book genre and building from there. The image for the cover was also created using generative AI using prompts suited of the genre and book title.

Project caption: The power of AI.
